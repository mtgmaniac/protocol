PROTOCOL & REROLLS

Gain +1 Protocol after each END TURN (cap 10). Spend from the strip:
Reroll — 2 pts. Tap the dice icon, then a hero die to reroll it.
Nudge +5 — 1 pt. Tap the arrow icon, then a hero to add +5 to their current roll.

ITEMS

Up to 3 consumable slots. After winning a fight you draft one item if you have room.
Use cost by rarity: Common 1 · Uncommon 2 · Rare 3 · Legendary 5 Protocol.
Tap an item slot to select it, then tap a target on the board.

XP & EVOLUTION

Tier-1 heroes earn XP after each battle win based on their average d20 rolls that fight: low avg → 2 XP · mid → 5 XP · high → 8–15 XP.
When a living tier-1 hero reaches 18 XP after battle 3 or later, an Evolution prompt appears. Choose a specialisation path — it replaces their ability kit, raises max HP, and locks them at tier 2 (no further XP).
Dead heroes do not earn XP. The XP bar below each hero portrait tracks progress.

HERO BADGES

Incoming = queued effects resolving this END TURN. Status = already active on the unit. The clock icon means more than 1 turn remains — the number is turns left.

Bolt (Incoming) — Damage hitting this hero from an enemy this turn.
Plus (Incoming / Status) — Incoming: heal arriving this turn. Status: active regen or heal-over-time.
Shield (Incoming / Status) — Absorbs damage before HP. Clock = turns until it expires.
Skull (Incoming / Status) — DoT (damage over time). Ticks once per player turn. Clock = ticks remaining.
Die — green (+) or red (−) — Net roll modifier on the hero's next d20. Green = buff; red = debuff. Incoming shows queued; Status shows what's already active.

ENEMY BADGES

Enemy cards show what heroes are doing to them (Incoming) and their own active effects (Status). Same clock rule — number = turns left.

Bolt (Incoming) — Total hero damage landing on this enemy this END TURN.
Red Die (Incoming) — Roll penalty (RFE) about to be applied to this enemy's d20.
Skull (Incoming) — DoT a hero is applying to this enemy this turn.
Plus (Incoming) — Lifesteal: this enemy heals itself when it hits. Value shown is the heal it would gain next hit.
Shield (Status) — Active shield absorbing the next hit(s). Clock = turns remaining.
Red Die (Status) — Active RFE debuff: this number is subtracted from the enemy's d20 each turn. Clock = turns left.
Green Die (Status) — Active roll buff: this number is added to the enemy's d20. Clock = turns left.

STATUS RIBBON

Word chips next to the HP bar. Tap any chip for a full description in-game.

Heroes
COWER — Hero skips rolling and abilities this round. Inflicted by certain enemy abilities.
CLOAK — ~80% chance the next enemy hit that targets this hero misses; then cloak clears.
TAUNT — All enemies target this hero this round. Triggered by hero abilities.
CURSED — Next roll this hero makes is rolled twice and the lower result is kept. Clears after.
SEAL N — Abilities in the named zone (e.g. STRIKE) fizzle. N = player rounds remaining.
Enemies
RAMPAGE N — Next N direct hits deal ×2 damage; one charge consumed per hit.
PACK +N — Deals +N bonus damage when striking; N = number of living packmates of the same type.
FOCUS — Heroes must target this enemy next player round. Cleared if the enemy dies.
FREEZE N — Die is locked for N tray reveals; the frozen face is preserved and the enemy can't roll higher.

BALANCE SIM

Monte Carlo — random 3-hero squads vs bundled encounter tables. Includes XP, evolution, DoT, and shield. No items or summons. Max 2 000 runs (browser). Copy output for spreadsheet analysis.
